## 📊 功能特性

### 筛选功能
- **全部**: 显示所有策展
- **对比反差**: 只显示对比类型的策展
- **认知颠覆**: 只显示认知类型的策展
- **趋势洞察**: 只显示趋势类型的策展

### 排序功能
- **最新**: 按日期降序排列
- **产品最多**: 按包含产品数量排序

### 统计信息
- 显示当前筛选结果数量
- 显示总策展数量
- 提供清除筛选快捷操作

## 📱 用户流程

1. 用户在发现页看到"今日 AI 策展"区块（显示前2条）
2. 点击"查看全部"按钮
3. 跳转到 `/discover/curations` 页面
4. 可以使用筛选器按类型筛选
5. 可以切换排序方式
6. 点击策展卡片中的产品可跳转到产品详情
7. 点击"我能做类似的吗？"按钮跳转到 AI 助手
