---
name: compare-products
description: 对比多个产品的各项指标
---

对比以下产品: $ARGUMENTS

请执行以下分析:

1. **基础信息对比**
   - 如果有产品 ID，使用 get_startups_by_ids 查询
   - 如果只有名称，使用 search_startups 查询
   - 列出名称、类目、描述

2. **财务指标对比**
   - 30天收入
   - 增长率
   - 估值倍数
   - 要价

3. **市场定位分析**
   - 目标客户群体
   - 竞争优势
   - 市场机会

4. **投资建议**
   - 综合评分
   - 风险因素
   - 推荐理由

以表格形式呈现对比结果，并给出最终推荐。
