---
name: market-overview
description: 生成指定类目或整体市场的概览报告
---

生成市场概览报告: $ARGUMENTS

请执行以下分析:

1. **市场概况**
   - 使用 mcp__saas__get_trend_report 获取整体趋势
   - 如果指定了类目，使用 mcp__saas__get_category_analysis

2. **头部产品**
   - 收入 Top 5 产品
   - 增长最快的产品
   - 最佳投资机会（低倍数）

3. **类目分析**
   - 各类目产品数量
   - 各类目平均收入
   - 热门类目趋势

4. **投资洞察**
   - 蓝海市场机会
   - 红海市场警示
   - 独立开发者适合的方向

以清晰的结构呈现，包含具体数据支撑。
