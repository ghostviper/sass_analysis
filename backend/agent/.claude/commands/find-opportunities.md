---
name: find-opportunities
description: 发现适合独立开发者的产品机会
---

发现产品机会: $ARGUMENTS

请执行以下分析:

1. **筛选条件**
   - 使用 browse_startups 浏览产品
   - 关注低技术复杂度、高收入潜力的产品

2. **机会评估**
   - 市场规模和增长
   - 竞争程度
   - 技术门槛
   - 启动成本

3. **成功案例**
   - 类似产品的成功故事
   - 可复制的模式

4. **行动建议**
   - 推荐的产品方向
   - 具体的切入点
   - 预期时间和投入

以独立开发者视角提供实用建议。
