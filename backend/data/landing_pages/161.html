Frequently Asked Questions

  • Is FormNX suitable for non-technical users?

    Yes, Absolutely! FormNX is designed for non-technical users. You don't need any coding skills to create professional-looking forms. The drag-n-drop interface make it easy to build forms without the need for any programming. You can also checkout out helpful documentation and tutorials to guide you through the process. If you encounter any difficulties our support team is readily available to assist you.

  • Can I embedd FormNX Forms on My Website?

    Yes, FormNX offers a lot of different options to embedd forms you created in your website. Just go to Publish tab, copy the code & use it. You can either embedd forms as iframe, link, feedback form or a popup.

  • How can i remove FormNX branding from forms?

    Yes, you can remove FormNX branding from forms by purchasing a premium package. Premium packages also offers additions benefits like increasing submissions limits, custom css/js, more workspaces & lot more. For price checkout the pricing page.

  • Does FormNX support multi-page forms?

    Yes, FormNX provides various option for multiple page forms, like one question per page or multiple question per page. This feature is particularly useful for longer forms or forms that require a step-by-step approach. You can divide your form into multiple pages, with each page containing a different set of fields. This not only improves the user experience by breaking down the form into manageable sections but also allows you to gather information progressively & doesnt overwhelms the user with a lot of fields.

  • Can I export form submissions for further analysis?

    Yes, FormNX allows you to export form submissions in various formats. You can export the data as CSV (Comma Separated Values) or Excel files, or PDF, making it easy to pick a relevant format, & import the data into other applications to perform data analysis tasks or keep data in sync.

  • Can I use FormNX with my team?

    Yes. You can create different workspaces, invite your team members & collaborate with them. This will also offer different permissions level for sharing of data & allowing form changes.
