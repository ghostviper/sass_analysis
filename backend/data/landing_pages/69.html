Sample Projects

Scientific Calculator

Keyboard-driven calculator with history log and responsive UI.

Use number keys and Enter to evaluate. History is kept in the right panel. Accessible labels are provided for screen readers.

Kanban Board

Drag and drop tasks across columns with keyboard and mouse.

Press Tab to focus cards; use arrow keys to move. Columns are landmarks for assistive tech.

SaaS Landing Page

Dark-themed landing with hero, features, and call to action.

Responsive layout with semantic landmarks and contrast-compliant colors.

Accessible Todo App

Todos with ARIA labels, keyboard shortcuts, and persistence.

Press Enter to add, Space to toggle, and Delete to remove. State is saved to localStorage.

Weather Dashboard

Real-time weather display with city search and 5-day forecast.

Enter a city name to fetch weather data. Uses OpenWeatherMap API with fallback mock data for demo.

Pomodoro Timer

Productivity timer with work/break cycles and session tracking.

Click Start to begin a 25-minute work session. Timer auto-switches to 5-minute breaks. Tracks completed sessions.
